# Tabel_and_Login — Tugas Pertemuan 2

Repository ini berisi tugas **Pertemuan 2** tentang dasar **HTML**, yaitu membuat:

## 1) Tabel Nilai
Halaman yang menampilkan **tabel nilai** menggunakan elemen HTML seperti:
- `<table>`, `<tr>`, `<th>`, `<td>`
- Penggunaan header tabel dan isi data (contoh: nama, NIM, nilai, dll.)

## 2) Login Page
Halaman **login** sederhana menggunakan elemen form HTML seperti:
- `<form>`
- `<input>` (username/email dan password)
- `<button>` / `<input type="submit">`

## Tujuan
Memahami struktur dasar HTML untuk:
- Menyusun data dalam bentuk tabel
- Membuat form input sederhana untuk login

## Cara Menjalankan
Buka file `.html` langsung di browser (double click), atau klik kanan **Open with Live Server** (jika memakai VS Code).
